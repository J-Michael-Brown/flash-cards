JavaScript

JavaScript is a programming language used to make web pages interactive. Like HTML and CSS.

Operators

An operator is a special character (or characters) that indicates an action to be performed. Some of them are: +, -, *, /, and %>

Variables

Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.

Always use the var keyword when initializing variables in JavaScript.

Variable Naming Convention

Lower Camel Case:

Use lowerCamelCase when naming JavaScript variables. Start with a lowercase letter, and if the variable name is more than one word, remove all spaces and capitalize the first letter of each subsequent word.

Function

A function is a block of code that performs an action and returns a result; optionally takes arguments

Methods

A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Argument

Some methods take one or more arguments that provide the method with additional information to help it perform its action.

EX: funtion add(number1, number2) {return number1 + number2;}; add(1, 2);

number1 and number2 are arguements, 1 and 2 are parameters.

Parameters

Variable that's assigned to the argument.

EX: funtion add(number1, number2) {return number1 + number2;}; add(1, 2);

number1 and number2 are arguements, 1 and 2 are parameters.

Return

The response to a method.

EX: >4832.78.toExponential();

"4.83278e+4"

The ">...()" indicates a method being performed. The following line is the return value, the response to the method.

Chaining Methods

Calling a method directly on the return value of another method.

Strings

A JavaScript data type that represents the exact text of whatever is enclosed in the quotes.

A string may contain letters, numbers, and charecters like !, 1, and 'ate'

Boolean

True and false are booleans. They are not strings - they simply represent being true or false.

Undefined

An object (such as a variable) without a defined value.

NaN

Not a Number. Considered a number by the computer, but unusable. EX: 0/0.

Escape

EX: > "Quoth the raven, \"Nevermore.\"";

The \\ around either side of the text above. It tells JavaScript that the " that comes right after it is not the end of the string, but just a character inside the string.

Alert

Opens a dialog box and returns the arguement.

ES: alert("hello world") will prompt a pop-up message on the page to say "hello world".

Comment

Anything following // on a line in JavaScript is ignored by the computer. Comments are a convenient way to leave notes in your code for yourself or other programmers.

jQuery

An extensive JavaScript library to help make our webpages interactive.

Attributes

Additional information provided to an HTML tag. For instance, the href attribute in an tag provides the URL a link should travel to.
